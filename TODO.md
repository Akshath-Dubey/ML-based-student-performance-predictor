# Student Performance Predictor

ML-powered Streamlit dashboard for predicting student performance based on behavioral features.

## 🚀 Live Demo

Deploy on [Streamlit Community Cloud](https://streamlit.io/cloud):

1. Go to [share.streamlit.io](https://share.streamlit.io)
2. Connect your GitHub account
3. Click **"New app"**
4. Select repository: `Akshath-Dubey/ML-based-student-performance-predictor`
5. Main file path: `app.py`
6. Click **Deploy**

## 📁 Files

| File               | Description                                                |
| ------------------ | ---------------------------------------------------------- |
| `app.py`           | Streamlit dashboard (9 graphs, ML accuracies, predictions) |
| `AI-Data.csv`      | Dataset                                                    |
| `requirements.txt` | Python dependencies                                        |

## 🖥️ Run Locally

```bash
# Activate virtual environment
student_env\Scripts\activate

# Run app
streamlit run app.py
```

**URL:** http://localhost:8501

## ✨ Features

- 9 Interactive Visualizations
- ML Model Accuracies (Decision Tree, Random Forest, Perceptron, Logistic Regression, MLP)
- Live Prediction Form + Recommendations
